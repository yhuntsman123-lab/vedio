# SEO + GEO Frozen Playbook

## Non-negotiable
1. Every public drama page must include schema.org VideoObject JSON-LD.
2. Every page must have canonical URL, OpenGraph, and rich text synopsis.
3. Provide machine-readable script summary + character tags + scene tags.
4. Submit sitemap + IndexNow ping for new public pages.
5. Track Bing AI Performance metrics and iterate on citation-ready formatting.

## Page Template Requirements
- H1 title with episode intent
- 120-180 word human-readable summary
- cast cards (character consistency anchors)
- key moments timeline
- transcript excerpt

## GEO Content Rules
- People-first writing; no keyword stuffing
- unique context per episode
- include production metadata (style, runtime, language)
- make important facts explicit for AI extractors
